# Corporate Intelligence Console System

## Direction
Dense, precise, secure. A tool for intelligence professionals. High contrast, void backgrounds, structural borders.

## Depth Strategy
**Borders-only.** No shadows. Hierarchy is defined by border opacity and subtle background lightness shifts.
- Base: Void Black (`#000000` or very dark zinc)
- Surface 1: `#09090b` (zinc-950)
- Surface 2: `#18181b` (zinc-900)
- Border Muted: `rgba(255,255,255,0.1)`
- Border Active: `rgba(255,255,255,0.2)`

## Spacing
Base unit: `4px` (0.25rem).
- Compact: `8px` (gap-2)
- Standard: `16px` (p-4)
- Section: `32px` (gap-8)

## Typography
- Headings: Sans-serif, tracking-tight.
- Data/Metadata: Monospace (`font-mono`), smaller size, uppercase for labels.
- Body: Sans-serif, high legibility.

## Key Patterns
- **The Data Beam:** Vertical line connecting related items.
- **Intelligence Blocks:** Chat messages are structured blocks with metadata headers.
- **Control Deck:** Sidebar uses same background as main, separated by border.
